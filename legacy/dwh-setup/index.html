Installation und Update des AKTIN-Applikationsserver

Diese Informationen richten sich an die System-Administratoren, die für die Einrichtung bzw. Wartung des AKTIN-Data Warehouse zuständig sind. Auf diesen Seiten wird das Vorgehen für eine Neuinstallation, für das Update und andere optionale Konfigurationsschritte beschrieben. Der Server für den Betrieb des AKTIN Data Warehouse sollte sich im Verantwortungsbereich des Klinikums befinden.

Folgende Hardware wird für den AKTIN Server empfohlen:

Kerne RAM Festplattenspeicher
Empfohlen 2 16 GB 100 GB
Minimum 1 8 GB 50 GB



Installation des Linuxservers

Für den Betrieb des AKTIN-Applikationsservers benötigen Sie eine funktionierende Installation eines auf Linux basierenden Server-Betriebsystems. Es wird empfohlen, das von uns geteste Ubuntu Server 20.04 LTS einzusetzen. Bitte beachten Sie die Seite Server-Installation für eine schrittweise Durchführung der Installation.

Neuinstallation des AKTIN-Applikationsservers

Besteht bereits ein laufender Server oder wurde das Server-Betriebssystem neu installiert, müssen Sie die Installation des AKTIN-Applikationsservers vornehmen. Eine Anleitung mit den durchzuführenden Schritten finden Sie auf der Seite AKTIN Installation. Sie erhalten mit einer Neuinstallation automatisch den aktuellesten Stand von Version 1.6rc1.

Update von DWH Version 1.3 oder 1.4

Mit der Veröffentlichung der Version 1.5 wurde der Installationsprozess von einem .tar.gz-Archiv zu einem Debian Paket umgestellt. Für ältere Versionen ist daher eine Migration auf die Debian-Paketierung notwendig. Zu diesem Zweck wird ein Migrationsskript bereitgestellt. Folgen Sie hierzu der Anleitung auf der Seite Update von der Version 1.3/1.4.

Automatisches Update des DWH

Seit der Version 1.5 des AKTIN Data Warehouse und der damit verbundenen Debian-Paketierung ist jederzeit eine automatische Aktualisierung über die Konsole oder über die Ansicht des Data Warehouse möglich. Folgen Sie hierfür den Hinweisen auf der Seite Automatisches Update

AKTIN-Software-Support

Sollten Sie bei der Installation, dem Update oder dem Betrieb der AKTIN-Software auf Probeleme stoßen, kontaktieren Sie den AKTIN-Support unter it-support@aktin.org.
